# Система учёта абитуриентов

Веб-приложение на Python для регистрации, поиска и управления списком абитуриентов.

## Возможности

- Добавление нового абитуриента
- Редактирование собственных записей
- Удаление собственных записей
- Поиск по имени, фамилии и номеру группы
- Сортировка по:
  - имени
  - фамилии
  - группе
  - баллам ЕГЭ
- Пагинация
- Авторизация по токену владельца записи
- Режим администратора с полным доступом ко всем записям
- Интерфейс на Bootstrap 5

---

- Python 3.14
- SQLite
- Jinja2
- python-dotenv

## Запуск без Docker

### 1. Клонировать репозиторий

```bash
git clone <URL_РЕПОЗИТОРИЯ>
cd student_list
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
```

### 3. Активировать окружение

Linux/macOS:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

### 4. Установить зависимости

```bash
pip install -r requirements.txt
```

### 5. Создать файл `.env`

Пример:

```env
DATABASE_NAME=test.db
SUPERUSER_TOKEN=admin
```

### 6. Запустить приложение

```bash
python app.py
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:8000
```

## Запуск через Docker

### Сборка образа

В корне проекта выполнить:

```bash
docker build -t student-list .
```

### Запуск контейнера

```bash
docker run -p 8081:8000 student-list
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:8081
```

## Администратор

Для входа администратора откройте страницу:

```text
/admin-login
```

и введите токен администратора, указанный в переменной окружения:

```env
SUPERUSER_TOKEN
```

После успешного входа администратор получает возможность редактировать и удалять любые записи. Для выхода из режима суперюзера добавьте к ссылке /logout

---

## Особенности Docker-версии

В Docker-образ включена демонстрационная база данных с тестовыми записями.

Обратите внимание:

- изменения, сделанные внутри контейнера, не сохраняются после его удаления;
- при повторном запуске создаётся новый контейнер с исходным состоянием базы данных.
